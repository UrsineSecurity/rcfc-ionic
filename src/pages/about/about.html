About Welcome to RCFC. This is a companion app to servers running the RCFC software.

To get started with a RCFC server, you just need Python 3.6+ on any computer. Then, pip install rcfc, and you are off to the races.

RCFC is maintained by Pat Viafore (@PatViaforever). You can find the source code for this phone app at GitHub or the source code for RCFC at GitHub. If you have a feature request or a bug, please log it there.
